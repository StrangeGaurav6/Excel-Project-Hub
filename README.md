An interactive MIS Dashboard built in Excel to analyze business performance using Sales, HR, Inventory, and Finance data.

- <a href="https://github.com/StrangeGaurav6/Excel-Project-Hub/blob/main/project%20excel.xlsx">Dataset view</a>

Slicers for dynamic filtering
XLOOKUP / VLOOKUP
KPI Cards (Sales, Target, Achievement %)
Conditional Formatting
